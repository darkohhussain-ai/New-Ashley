# **App Name**: Ashley DRP Manager

## Core Features:

- Data Backup: Save the current state of the application including lists, edits, logo, and fonts to the browser's local storage.
- Data Export: Export the application data to a file for offline storage.
- Data Import: Import application data from a previously saved file.
- Customizable Logo: Upload and change the application logo displayed on the intro screen and login dashboard.
- Volunteers HR Section: Manage volunteer human resources with data persistence using file storage.
- Settings Panel: Allow users to change design colors, application fonts, enable dark mode and save preferences locally.
- Admin Panel: Grant access to Darko with password '123456' and admin level priveledges.

## Style Guidelines:

- Color palette: Based on clarity and trustworthiness. Primary color: Light sky blue (#87CEEB) for a calming, organized interface.
- Background color: Very light gray (#F0F8FF) for a clean, unobtrusive background that emphasizes data.
- Accent color: Muted purple (#B098B5) to bring emphasis to icons and actionable items.
- Option for users to toggle to a Dark Mode version of the site from settings, which inverts the color scheme. Dark Mode scheme: Primary color: Deep cyan (#008080); Background color: Dark slate gray (#2F4F4F); Accent color: Lavender (#E6E6FA).
- Body and headline font: 'Inter' (sans-serif) for a clean, modern user experience. Note: currently only Google Fonts are supported.
- Modern, glass-style design with sections displayed as large icons with small text, inspired by Windows 11.
- Centered data layout, connected to file storage for persistence. Each section has customizable colors through the Settings.
- Use clear, simple icons for each section, reflecting their function, such as user icons for Volunteers HR, box icons for Item Placement, and arrow icons for Transmit.
